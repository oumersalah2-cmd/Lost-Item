## 📌 Project Overview

**CampusTrack** is a web-based platform that helps students and staff at Addis Ababa University report, search, and recover lost or found items on campus. It replaces inefficient physical notice boards and manual logs with a centralized, easy-to-use digital system.

## 🚀 Key Features

- 🔐 User registration & login (students and staff)
- 📋 Post lost item reports with photo, description, location
- 🔍 Post found item reports
- 🔎 Search & filter items by keyword, category, date, location
- 🔔 Notifications when a match is found
- 🛡️ Admin dashboard — moderate posts, manage users
- 📊 Item status lifecycle: Active → Claimed → Closed
