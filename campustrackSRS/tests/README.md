# 🧪 Tests

This folder contains all test cases and reports for CampusTrack.

**Owner:** Biruk Semaw (QA & Tester)

## Structure

```
tests/
├── unit/           # Unit tests for individual functions/modules
├── integration/    # Integration tests (API + database)
├── reports/        # Test result reports
└── README.md
```

## Files to be added:
- `unit/` — Unit test files
- `integration/` — Integration test files
- `reports/test_report.md` — Final test results and bug log

## Testing approach:
- Unit testing: test individual backend functions
- Integration testing: test API endpoints with Postman / automated
- User Acceptance Testing (UAT): manual testing of all user flows
