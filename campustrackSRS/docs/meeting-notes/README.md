# 📝 Meeting Notes

This folder stores weekly team meeting notes.

## Naming convention:
`YYYY-MM-DD_meeting.md`

## Example:
`2026-05-01_meeting.md`

## Template:

```
# Team Meeting — [Date]

**Attendees:** [names]
**Duration:** [e.g. 1 hour]

## Agenda
1. ...

## Discussion
- ...

## Decisions Made
- ...

## Action Items
| Task | Assigned To | Due |
|------|-------------|-----|
| ...  | ...         | ... |

## Next Meeting
[Date and time]
```
